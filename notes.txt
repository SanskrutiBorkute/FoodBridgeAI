Project Overview
----------------------------------------------------------

FoodBridgeAI is an AI-powered web application that helps reduce food wastage by connecting food donors with NGOs.

The platform allows donors to register surplus food, NGOs to register themselves, and uses Machine Learning models to predict food demand and prioritize NGOs for efficient food distribution.

Frontend is deployed on Netlify.
Backend is deployed on Render.

How the Project Works
----------------------------------------------------------

1. Donor opens the website.

2. Donor fills donation details.

3. Frontend sends data to FastAPI backend.

4. Backend validates the data.

5. Data is stored inside SQLite database.

6. NGO information is stored.

7. AI Priority Model calculates NGO priority.

8. Forecast Model predicts food demand.

9. Dashboard displays all information.

How to Deploy Backend on Render
----------------------------------------------------------

1. Push project to GitHub.

2. Login to Render.

3. Create New Web Service.

4. Connect GitHub Repository.

5. Select backend folder.

6. Install dependencies automatically.

7. Deploy.

8. Copy Backend URL.

----------------------------------------------------------
How to Deploy Frontend on Netlify
----------------------------------------------------------

1. Push project to GitHub.

2. Login to Netlify.

3. Import Project.

4. Connect GitHub.

5. Select Repository.

6. Build settings

Base Directory:
Leave Empty

Publish Directory:
Leave Empty

7. Deploy.

8. Copy Netlify URL.

How to Update the Website
